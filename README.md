# MySpider
========================
与爬虫相关的个人工具集  
Personal toolset for crawler
#### Download
Windows version：[MySpider-x64-v0.1.zip](https://pan.baidu.com/s/14emt_SMXvBsZ2nKMBTlrIQ)

#### Build
1. run venv.sh or venv.bat
2. run init.sh or init.bat
3. run release.sh or release.bat
